# Интеграция с Telegram

## Структура приложения

### Страницы
- `register.html` - Регистрация новых пользователей
- `admin.html` - Панель администратора для одобрения заявок
- `index.html` - Главная страница (после одобрения)
- `client.html` - Интерфейс клиента
- `trainer.html` - Интерфейс тренера

## Процесс регистрации

1. Пользователь открывает бота в Telegram
2. Нажимает кнопку "Открыть приложение" (меню внизу)
3. Открывается `register.html`
4. Выбирает роль (клиент или тренер)
5. Отправляет заявку
6. Администратор получает уведомление
7. Администратор одобряет/отклоняет через `admin.html`
8. После одобрения пользователь получает доступ

## Настройка Telegram бота

### Команды для BotFather:

```
/newbot - создать нового бота
/setmenubutton - настроить кнопку меню
/setcommands - настроить команды
```

### Список команд:
```
start - Начать работу с ботом
register - Зарегистрироваться в приложении
profile - Мой профиль
workouts - Мои тренировки
help - Помощь
```

### Кнопка меню:
- Текст: "Открыть приложение"
- URL: https://your-domain.com/register.html

## База данных

### Таблица user_registration
- TelegramId - ID пользователя в Telegram
- Username - username в Telegram
- FirstName, LastName - имя и фамилия
- RequestedRole - запрошенная роль (client/trainer)
- Status - статус заявки (pending/approved/rejected)
- ApprovedRole - одобренная роль
- RegistrationDate - дата регистрации

## Роли пользователей

### Администратор
- URL: `admin.html`
- Доступ: прямая ссылка https://your-domain.com/admin.html
- Функции: 
  - Одобрение/отклонение заявок
  - Назначение ролей (клиент или тренер)
  - Уведомления о новых заявках
  - Кнопка "Админ" в навигации для быстрого доступа

### Клиент
- URL: `client.html`
- Функции: тренировки, измерения, профиль

### Тренер
- URL: `trainer.html`
- Функции: управление клиентами, упражнения, программы
