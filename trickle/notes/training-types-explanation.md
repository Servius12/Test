# Типы Тренировок - Объяснение

## Основные Принципы

### Обычная Тренировка (Standard)
- Выполняется упражнение за упражнением
- **Отдых после КАЖДОГО упражнения** (обычно 60-90 секунд)
- Классическая схема: упражнение → отдых → упражнение → отдых

### Круговая Тренировка (Circuit Training)
- Несколько упражнений выполняются последовательно
- **Отдых ТОЛЬКО после завершения всего круга**
- Схема: упражнение 1 → упражнение 2 → упражнение 3 → ОТДЫХ → повторить круг
- Обычно 3-5 кругов

### Суперсет (Superset)
- **Соединение 2-3 упражнений** в одну группу
- Упражнения выполняются **БЕЗ отдыха между ними**
- Отдых только после завершения всех упражнений в суперсете
- Схема: упражнение 1 → упражнение 2 → ОТДЫХ → повторить

### Дропсет (Drop Set)
- Применяется к **ОДНОМУ конкретному упражнению**
- Выполняется до отказа, затем вес снижается на 20-30%
- Продолжение выполнения **БЕЗ отдыха**
- Схема: 10 повторов с 20кг → сразу 8 повторов с 15кг → сразу 6 повторов с 10кг → ОТДЫХ

### AMRAP (As Many Reps As Possible)
- Выполнить максимум повторений/кругов за определенное время
- Время: обычно 5-20 минут

### EMOM (Every Minute On the Minute)
- Каждую минуту выполняется определенное упражнение
- Оставшееся время в минуте = отдых

## Важно для Реализации

1. **Обычная тренировка** - отдых после КАЖДОГО упражнения
2. **Круговая** - отдых только после КРУГА упражнений
3. **Суперсет** - это ГРУППА упражнений, не тип всей тренировки
4. **Дропсет** - это ТЕХНИКА для одного упражнения, не тип тренировки
